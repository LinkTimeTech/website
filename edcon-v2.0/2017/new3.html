Blockchain Talent Database

Blockchain technology is an interdisciplinary technology. It not only includes cryptography, big data and many other fields in computer science, but also requires developers to deeply understand finance, law and management. The shortage of talents with the interdisciplinary experience is hindering the blockchain industry's development. Figuring out how to gather international talents is a very big challenge for the industry, so this conference will set up a Talent Database to strongly support companies' needs for talents.
