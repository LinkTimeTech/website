uRaiden Payment Channels Launches on the Main Ethereum Blockchain Network

“As one of the recently fastest-growing digital assets, a streamlined version of Ethereum’s Lightning Network, uRaiden, has been officially launched on main Ethereum blockchain yesterday”

A streamlined version of ethereum’s Lightning network, μRaiden (also as Micro Raiden), was officially activated yesterday, several months since its first launch on ethereum’s test network.

As a streamlined version of ethereum’s Raiden network currently under development, uRaiden aims to provide a mechanism for micropayments similar to bitcoin’s Lightning network. Same as the Raiden network, uRaiden enables mutually-trusted, free and instant transactions between two parties, and builds a many-to-one payment channel system applicable to both. However, it’s not applicable to a many-to-many payment environment for it requires users to predetermine the tokens sent to potential receivers. This limitation allows for the use of uRaiden on ethereum’s main net by making the technology less complex.

Like bitcoin’s Lightning network, uRaiden scales the ethereum network by building a second layer that would exist above the main blockchain. With the second layer network, the participants could create payment channels which would reduce transaction costs.

As for the reason why “uRaiden” is developed, the development team explained that it aimed to build tools for decentralized APP (dapp) developers to open up payment channels without some of the complex features planned for the whole Raiden network.

The team said, “Talking to dapp developers, we noticed that many of them just want to use the Raiden Network as a robust many-to-one payment channel system; one service provider offering services to many recurring customers.”

Developer Lefteris Karapetsas wrote on GitHub yesterday, “The version being released is aimed at sussing out any bugs in the code, and a limit has been placed on the maximum amount of Raiden tokens (RDN) that can be deposited for making payments. To that end, we have limited the maximum deposit to 100 RDN.”
