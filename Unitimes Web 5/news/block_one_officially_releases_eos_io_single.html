Block.One Officially Releases EOS.IO Single-Thread Application Test Network (EOS STAT)

“Following a new record Block.One set by raising $185 million in the ICO of its EOS in July this year, the company officially released EOS.IO single-thread application test network EOS STAT yesterday.”

Block.One, a blockchain startup, officially released EOS.IO single-thread application test network EOS STAT yesterday. According to the report, the release was ahead of schedule, which allows internal teams and the developer community to ensure that EOS.IO could be successfully released on June 1, 2018.

EOS.IO is a blockchain operating system that provides accounts, authentication, databases, asynchronous communication and the scheduling of applications across multiple CPU cores or clusters. EOS STAT imitates the EOS.IO software with 21 block production nodes, one block-out time, a stable developer API, a wallet for setup, an account system using the API, contract security and verification.

The release of EOS STAT enables developers to assess, build and test decentralized applications in a common environment, and allows potential block producers to evaluate EOS.IO. Daniel Larimer, CTO of Block.One, said, “We are excited to launch this test network, and we will continue to improve the performance of EOS.IO in the hope of receiving feedback from the developer community.”

It is revealed that EOS.IO will release its Multi-Thread Application Test Network (MAT) in 2018 and MAT will be refactored and will have blockchain communication capabilities.

“Even though on a single-thread test network, EOS.IO now reaches TPS performance of over 1,000, which makes it one of the fastest smart contract environments ever.” Daniel Larimer added at last.
